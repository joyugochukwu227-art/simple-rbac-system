# Simple RBAC System

Description
This project is a simple Role-Based Access Control (RBAC) system that assigns permissions to users based on their roles.

## Roles and Permissions
- Admin → create, read, update, delete
- Editor → create, read, update
- Viewer → read only

 How it Works
The system checks a user's role and determines whether they are allowed to perform a specific action.
